Detailed Description

TUFLOW FV format.

Binary NetCDF format with structure similar to UGRID stored as 3D Layered Mesh (https://github.com/qgis/QGIS-Enhancement-Proposals/issues/158)

Supports active flag.

There are special datasets for maximum/minimum/time of maximum/time of minimum. These datasets are not 1-1 with calculated max/min of time-datasets, since they are calculated with more fine methods directly in TUFLOW solvers

Both mesh and dataset is stored in single file.
