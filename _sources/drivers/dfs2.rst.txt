.. _driver.dfs2:

================================================================================
DFS2 -- DHI Grid Series File
================================================================================

.. shortname:: dfs2

MDAL supports reading of DFS2 format. DFS2 is DHI Grid Series File defined by DHI_.
This is external driver that is not shipped directly with MDAL installations.
To build the driver, follow instruction in README DFS2_README_

Driver is available only on Windows.

.. _DHI: https://www.dhigroup.com
.. _DFS2_README: https://github.com/lutraconsulting/MDAL/tree/master/external_drivers/dhi_dfs
